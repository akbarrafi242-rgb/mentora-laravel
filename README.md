# Mentora - Platform Mentoring Perlombaan Mahasiswa

## Deskripsi
Mentora adalah platform yang menghubungkan mahasiswa dengan mentor berpengalaman di bidang perlombaan. Pengguna dapat memilih mentor sesuai kebutuhan perlombaan mereka.

## Fitur CRUD yang Diimplementasikan
- **Create** - Register akun pengguna baru
- **Read** - Menampilkan data profil pengguna
- **Update** - Mengupdate profil (username, email, no. telepon, bio, password)
- **Delete** - Menghapus akun pengguna

## Tech Stack
- **Framework**: Laravel 12
- **Language**: PHP 8.2
- **Database**: MySQL
- **CSS Framework**: Tailwind CSS
- **Local Server**: XAMPP
- **Build Tool**: Vite

## Cara Menjalankan Project
1. Clone repository ini
2. Copy `.env.example` menjadi `.env`
3. Setting database di `.env`
4. Jalankan `php artisan key:generate`
5. Jalankan `php artisan migrate`
6. Jalankan `npm install` lalu `npm run dev`
7. Jalankan `php artisan serve`
8. Buka `http://localhost:8000`
